# 循环思考增强版（v3.0 Ask Your Question）

基于 SIO-MMOS 方法论的循环深度思考流程，具备**全生命周期主动协作能力**、**专业工具自动选择能力**和**苏格拉底式提问能力**。

---

## 🆕 v3.0 核心能力（Ask Your Question）

### 1. 苏格拉底式提问引擎
- **目标**: 通过深度提问澄清需求、暴露假设、挖掘真实意图
- **方法**: 采用苏格拉底六问法（澄清→假设→证据→视角→影响→元问题）
- **触发**: 当任务清晰度 < 70% 时自动启动

### 2. 多轮对话协作框架
- **第一轮**: 方法论确认（A/B/C 模式选择）
- **第二轮**: 深度提问（苏格拉底六问）
- **第三轮**: 任务规格确认
- **后续轮**: 阶段性成果确认

### 3. 动态清晰度评估
- **实时计算**: 每个 Phase 结束后更新清晰度评分
- **阈值管理**:
  - 🔴 < 50%: 必须协作，暂停推进
  - 🟡 50-70%: 建议协作，关键问题确认
  - 🟢 > 70%: 可继续推进，记录假设

### 4. 假设追踪与管理
- **假设识别**: 自动识别推理中的隐性假设
- **假设分类**: 关键/次要/已验证/待验证
- **假设确认**: 关键假设必须经用户确认
- **假设记录**: 所有假设写入 assumptions.md


## 📋 完整执行流程

### ⚡ Phase 0: 增强任务理解（Ask Your Question）

#### Step 0.1: 方法论确认（第一轮对话）

**规则**: 在任何对话发起后的第一轮，必须与用户确认是否应用 SIO-MMOS 方法论。

**确认模板**:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🧠 **方法论选择**
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

我注意到您的问题："{问题摘要}"

请确认分析模式：

🔷 **A：SIO-MMOS 深度分析**（复杂问题，10-30分钟）
   - 五维系统分析（道法势术器）
   - 主客体关系梳理
   - 苏格拉底式深度提问
   - 递归迭代深度思考

🔶 **B：快速分析**（中等复杂度，3-5分钟）
   - 结构化分析
   - 核心要点提取
   - 快速建议

🔹 **C：直接回答**（最简模式）
   - 直接回答，不做深度分析
   - 1-2分钟完成

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💬 请回复 A、B 或 C
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

#### Step 0.2: 苏格拉底式深度提问（第二轮对话 - 如选A模式）

**触发条件**: 用户选择 A 模式（SIO-MMOS 深度分析）

**苏格拉底六问框架**:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔍 **苏格拉底式深度提问**
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

为了帮助您获得最有价值的分析，我需要深入理解您的需求。

**第一问：澄清性问题** (Clarification)
您提到 "{用户原始问题}"，能否更具体地描述：
- 您希望解决的核心问题是什么？
- 成功的结果会是什么样子？

**第二问：假设性问题** (Assumptions)
基于您的描述，我识别到以下潜在假设：
- 假设1: {推断的假设}
- 假设2: {推断的假设}
这些假设正确吗？还有哪些我没考虑到的关键假设？

**第三问：证据性问题** (Evidence)
- 您有哪些数据或信息支持这个方向？
- 有哪些反例或风险需要考虑？

**第四问：视角性问题** (Perspectives)
- 这个问题还涉及到哪些利益相关者？
- 他们可能有怎样不同的视角？

**第五问：影响性问题** (Implications)
- 如果成功解决这个问题，会带来什么连锁效应？
- 如果失败，最大的风险是什么？

**第六问：元问题** (Meta-Question)
- 您为什么会提出这个问题？
- 这个问题的根源是什么？

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
💬 请回答以上您愿意回答的问题，不需要全部回答
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```

**简化版提问（如用户时间有限）**:

```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔍 **核心问题澄清**
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

为了给您最精准的分析，请帮助我理解：

1. **目标**: 您最终希望达成什么结果？
2. **背景**: 为什么现在提出这个问题？
3. **约束**: 有哪些限制条件（时间/资源/技术）？
4. **成功标准**: 如何判断分析结果是成功的？

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
```
