# 业务/产品方案价值评估技能
## 技能定位
基于蜂群智能体三维评估框架，对业务方案、产品规划、项目提案进行多维度价值验证，判断是否对目标受众、决策主体及所有利益相关方产生真实价值，是否具备可落地性。
## 评估前置条件
执行评估前必须明确以下信息：
1. 方案名称与核心目标
2. 目标受众群体（终端用户、客户等）
3. 决策主体（投资方、管理层、项目负责人等）
4. 所有相关利益方（合作方、员工、第三方等）
5. 预期投入（人力、财力、时间成本）
6. 预期产出与收益目标
## 第一阶段：多角色独立评估
六个评估智能体独立输出评估报告，不得互相干扰：
### 1. 目标受众价值评估智能体
输出《目标受众价值报告》，包含：
- 方案解决了目标受众的什么核心痛点？
- 痛点的真实程度（1-10分）、受众覆盖范围
- 受众愿意为该方案付出的成本（金钱、时间、学习成本）
- 相比现有解决方案的差异化优势
- 受众接受度预测与潜在阻力
### 2. 决策主体价值评估智能体
输出《决策主体价值报告》，包含：
- 方案是否符合决策主体的战略目标与发展方向？
- 投入产出比（ROI）测算与回报周期
- 对决策主体核心指标的影响（营收、效率、市场份额、品牌等）
- 风险可控性与最坏情况评估
- 战略层面的机会成本与替代方案比较
### 3. 利益相关方价值评估智能体
输出《利益相关方价值报告》，包含：
- 所有相关方的价值增益与损失分析
- 各方的支持意愿与阻力点
- 多方利益平衡机制与冲突解决方案
- 生态协同效应与负面外部性评估
### 4. 落地可行性评估智能体
输出《落地可行性报告》，包含：
- 现有资源（人力、技术、资金、渠道）是否支撑落地？
- 关键路径与里程碑节点设计合理性
- 核心依赖与外部不确定性分析
- 潜在风险点与应对预案
- 最小可行版本（MVP）的可行性与验证路径
### 5. 长期价值评估智能体
输出《长期价值报告》，包含：
- 方案的生命周期与长期迭代潜力
- 可复用能力与边际成本递减效应
- 对组织能力、技术壁垒、品牌资产的长期增益
- 市场变化与技术演进下的适应性
- 退出机制与转型成本
### 6. 反方质询评估智能体
输出《风险与质疑报告》，包含：
- 方案的最大致命缺陷是什么？
- 哪些假设不成立会导致方案完全失败？
- 最坏情况发生的概率与影响程度
- 反对者的核心理由与合理性
- 方案的局限性与适用边界
## 第二阶段：交叉验证与对冲
必须完成以下交叉质询：
1. 目标受众价值 ↔ 决策主体价值 是否对齐？
2. 短期收益 ↔ 长期价值 是否平衡？
3. 理想预期 ↔ 落地可行性 是否匹配？
4. 收益评估 ↔ 风险评估 是否对称？
## 第三阶段：综合评估结论
输出最终《价值评估综合报告》，必须包含：
1. 综合价值评分（0-100分）
2. 明确的评估结论：强烈推荐/推荐/谨慎推荐/不推荐
3. 核心价值点清单（按优先级排序）
4. 核心风险点清单（按严重程度排序）
5. 落地建议与优化方向
6. 决策参考阈值（满足什么条件时应该推进/终止）
## 输出契约
- 所有评估必须基于事实与数据，禁止主观臆断
- 必须明确标注评估假设条件与信息缺口
- 对于不确定的内容必须明确说明置信度
- 报告必须结构化、可量化、可验证
