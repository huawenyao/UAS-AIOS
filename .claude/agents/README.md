# SelfPaw 蜂群认知智能体

> 基于否定之否定辩证升维的分布式认知对手盘体系

## 智能体清单

| 文件 | 智能体 | 立场 | 对手盘 |
|------|--------|------|--------|
| 01_user_perspective.md | 用户视角智能体 | 终端用户/需求方 | 买单方、决策 |
| 02_obstacle_guardian.md | 关卡障碍智能体 | 极致理性风险把控 | 决策、博弈观察 |
| 03_core_decision.md | 核心决策智能体 | 个人目标执行者 | 用户、关卡 |
| 04_value_buyer.md | 买单价值智能体 | 价值变现与成本把控 | 决策、用户 |
| 05_game_observer.md | 博弈观察智能体 | 第三方中立 | 无（统筹者） |

## 三阶段工作流

1. **初次否定**：五智能体并行独立输出，坚守立场
2. **二次否定**：认知对手盘博弈、质询、信息补全
3. **辩证融合**：整合合理内核，涌现《全维度辩证决策方案》

## 使用

```
/swarmDecision [决策议题]
```

详见 `.claude/commands/swarm_decision.md`
