# AI Recruitment OS Prototype

Run a local static server before opening the prototype because the page loads
`data/demo_state.json` through `fetch()`.

```powershell
python -m http.server 8090
```

Then open:

```text
http://localhost:8090
```

The prototype focuses on user-perceived value:

- Clear hiring mission
- Role success model
- Evidence-based shortlist
- Structured interview workspace
- Calibration decision room
- Candidate-facing transparency
- Post-hire learning loop

The demo data is generated by:

```powershell
python ..\scripts\agent_runtime.py --write-demo
```
