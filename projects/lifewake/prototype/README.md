# LifeWake 高保真交互 Demo

一个无构建依赖的“生命情景空间”：以关注的人为焦点，用可触摸、可移动的拟物对象策展今晚的惊喜。

## 启动

```bash
cd projects/lifewake/prototype
python3 -m http.server 8090
```

浏览器打开 <http://localhost:8090>。无需安装依赖，也不访问外部图片、字体或 CDN。

## Walkthrough

1. 首屏确认关注主体“妻子 · 林妍”和情景“今晚给她一个惊喜”。
2. 拖动桌面的相框、唱片机、礼物盒等物件，刷新前的位置会保留在当前 session。
3. 打开礼物盒，选择“月光晚餐”“城市寻宝”或“心跳唱片”。
4. 轻触唱片机播放/暂停模拟旋律；展开照片并选择共同记忆。
5. 用黄铜时钟决定惊喜时刻；在信封中写下自己的表达并封缄。
6. 用氛围灯切换暖光、月光、星光；打开手册查看“起、承、转、合”与完成度。
7. 点击“预演惊喜”，逐幕查看由所选对象组成的完整仪式，按 `Esc` 返回。
8. 从顶部切换关注主体与生命情景，观察标题、建议、色调和空间提示变化。
9. 打开“主权与边界”，查看材料、用途、受益者、期限，并可撤回本次材料。

## 检查

```bash
node --check app.js
python3 smoke_test.py
```

小屏（宽度不超过 760px）会重排为纵向移动情景桌面；所有对象仍可从桌面或底部 Dock 访问。
