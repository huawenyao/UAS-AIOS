# LifeWake 执行协议

1. **意图归一**：仅接受 `surprise_delivery` / `pulse_solo` / `pulse_duet` / `feedback_review`。
2. **同意先行**：调用任何创作能力前必须 `lw.consent.check`；purpose 仅 `create_for_user`。
3. **独特性**：惊喜必须含 `uniqueness_refs` 与 `inspiration_trace`。
4. **双向性**：`pulse_duet` 必须双方同意且 `needs_met` 覆盖全部参与者。
5. **仪式化**：交付前 `lw.ritual.render`，`wow_score >= threshold`。
6. **可撤回**：同意撤回立即中止队列；审计保留，原始生物流不落盘。
7. **慢灵感**：遵守 `max_surprises_per_day`。
8. **演化**：反馈只生成 `auto_apply: false` 的 ChangeSet。
