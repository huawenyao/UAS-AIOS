# 《全维度辩证决策方案》输出契约

最终输出必须包含以下字段：

- `topic`: 当前决策议题
- `final_decision`: 最终结论
- `execution_path`: 分阶段执行路径
- `risk_preplan`: 主要风险与预案
- `user_adaptation`: 用户适配与体验策略
- `cost_controls`: 成本、价值与买单阈值控制项
- `game_response`: 对外部博弈与变量变化的应对策略
- `consensus`: 蜂群共识
- `open_conflicts`: 尚待观察的冲突点
- `retrospective_triggers`: 触发复盘的信号
