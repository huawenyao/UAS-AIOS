# AI 招聘 · 竞品洞察与系统定位（2025–2026）

> 内外部洞察摘要，供产品与架构决策引用。详述见上一轮战略分析对话。

---

## 1. 第一性：招聘优化什么

在不确定信息下，用有限面试官时间，最大化录用质量、最小化错招与合规风险。AI 的最大杠杆在漏斗上段（sourcing → 初筛 → 结构化首轮），并把标准变成可审计的组织资产。

---

## 2. 外部格局（分场景冠军）

| 场景 | 领先方案 | 核心能力 |
|------|----------|----------|
| 企业人才战略 / 内部流动 | Eightfold、HiredScore | 技能图谱、Agentic TA、AI Interviewer |
| 大规模校招/蓝领 | HireVue、北森 AI 面试官 | 异步视频、结构化追问、7×24 |
| 候选人体验与调度 | Paradox、GoodTime | 对话预筛、自动排期 |
| 流程中枢 | Greenhouse、Ashby、Moka | 公平流程、看板、AI 插件 |
| 主动 sourcing | Gem、HireEZ、SeekOut | 外呼、被动候选人 |
| 端到端 Agent | HeyMilo、Alex、Tezi、AI-HR | 自主 sourcing→面试→回写 ATS |

**代际**：1.0 规则 ATS → 2.0 流程 SaaS + Copilot → 3.0 Applied/Agentic AI（多步自主 + 人在环）。

---

## 3. 中国市场

| 玩家 | 定位 |
|------|------|
| 北森 | 一体化 HR + AI Family（面试官、DeepSeek 等） |
| Moka | 招聘场景 AI 原生，Eva 智能体 |
| BOSS 直聘 | 流量撮合 + 南北阁模型 |
| Agent 插件型 | 世纪云猎等，补 ATS 前端断点 |

典型组合：**ATS + 1~2 专项 AI**，而非单点替换。

---

## 4. 本项目定位（`examples/ai-recruitment`）

### 4.1 应锚定场景

- **A. 中高端精招**（复杂 JD、证据链决策）— 高适配  
- **F. 合规敏感**（可配置标准、审计）— 高适配  

不优先与 Eightfold/北森做全栈对标；不与 HireVue 争批量视频测评首发。

### 4.2 已有差异化

- ASUI **知识即配置**（`evaluation_criteria.md`）  
- **证据链 + 风险标识** HTML 报告  
- **实体–事件闭环**（Job/Candidate/Task/Event/Evaluation）  
- **ReqHarness 契约**（`domain_world_model` + `recruitment_closed_loop`）

### 4.3 关键缺口

| 缺口 | 优先级 |
|------|--------|
| HR 生产入口（已补 `run_batch.py`） | P0 |
| 价值摘要产品化（已补 `value_summary.py`） | P0 |
| 候选人拒信话术接入主路径 | P1 |
| AI 结构化初面 | P2 |
| ATS webhook、录用回流 Learn | P2 |

---

## 5. 目标架构（一句话）

**决策大脑（知识层 + 证据）+ 执行 Agent 网格（Screener/Interviewer/Communicator）+ 渠道共生（ATS/BOSS）**，而非单体 ATS 克隆。
