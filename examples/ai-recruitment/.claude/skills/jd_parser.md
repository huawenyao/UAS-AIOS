# 岗位解析规则

## 提取维度

| 维度 | 说明 | 示例 |
|------|------|------|
| 学历 | 最低学历要求 | 本科/硕士/博士 |
| 工作经验 | 年限要求 | 3-5年 |
| 技能关键词 | 技术栈/工具 | Python, SQL, 数据分析 |
| 薪资范围 | 可选 | 20-30K |
| 工作地点 | 可选 | 北京/远程 |

## 标准化映射

- 学历：高中/大专/本科/硕士/博士 → 等级 1-5
- 经验：应届/1-3年/3-5年/5年+ → 等级 1-4
- 技能：提取原文关键词，保留顺序

## 输出 Schema

```json
{
  "education": "本科",
  "experience_years": "3-5",
  "skills": ["Python", "SQL", "数据分析"],
  "salary_range": "20-30K",
  "location": "北京"
}
```
