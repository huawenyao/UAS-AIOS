## 采集方法验证结果

### ✅ 成功方案：browser_use (Playwright)
- 成功绕过 GitHub 反爬虫机制
- 成功获取用户搜索结果
- 单页可采集约10位候选人

### ❌ 失败方案：Selenium + Chrome/Edge

- 反爬虫检测严格
- 需要额外配置代理和stealth模式
