# 数字人生态产品愿景（Harness 知识源）

> 权威战略文档：`docs/strategic/Enterprise_Digital_Human_Ecosystem_Product_Definition.md`

## 产品公式

```
企业级 AI = UAS 平台 × 模型层 × 数字基础能力 × 业务连接器 × 类人 Agent
```

## 双轨定位

| 产品 | 角色 | 核心场景 |
|------|------|----------|
| **SelfPaw 企业版** | 员工数字分身 | 对内效率、意图升级、岗位 Domain |
| **ΠPaw** | 经营数字岗位 | **经营向外**、多 Agent 协同、KPI 闭环 |

## 数字人三层

- **L1** 个人数字分身 → SelfPaw  
- **L2** 职能数字人 → ΠPaw 工作台  
- **L3** 经营数字人（对外）→ ΠPaw + Outward Gateway  

## Phase-0 优先级（B 端优先）

1. 企业数据平面 + 权限 + 审计  
2. `cs.*` 能力服务框架 + ≥5 语义服务  
3. CRM/OA 连接器 ≥2  
4. SelfPaw 企业版 S1-S2 + ΠPaw 客服 Agent 标杆  

## 成功指标（12 月）

- 能力服务 ≥15 · 连接器 ≥5  
- SelfPaw WAU 60% · 意图闭环 70%  
- ΠPaw 客服自动解决 40%  
- 审计覆盖 100%
