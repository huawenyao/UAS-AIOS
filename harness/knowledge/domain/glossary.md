# 数字人生态术语表

| 术语 | 定义 |
|------|------|
| **数字人生态** | 企业内 L1 个人分身 + L2 职能 Agent + L3 对外经营 Agent 的协同体系 |
| **User AGI** | SelfPaw：面向员工（组织内）的类人 Agent 产品轨道 |
| **Business AGI** | ΠPaw：面向组织经营（含对外）的类人 Agent 产品轨道 |
| **经营向外** | 获客、转化、履约、服务、伙伴协同等对外经营目标 |
| **能力服务 cs.\*** | 业务语义 API，Agent 执行业务的唯一入口 |
| **数据平面** | 租户、主数据、事件流、审计链的企业主权数据层 |
| **数字基础能力** | 流程、表单、权限、数据四类可产品化封装能力 |
| **岗位 Domain** | 行业/岗位认知包：Ontology + 约束 + 能力包 + 评估透镜 |
| **岗位 Agent 编制** | ΠPaw 中可启停的数字岗位（销售顾问、客服等） |
| **Intent Hub** | SelfPaw 意图识别与向 ΠPaw 升级的入口 |
| **Outward Gateway** | 对外统一触达与审计网关 |
| **五件套** | 对象-状态-事件-门禁-ChangeSet（AIPOS） |
| **八元组** | I,K,R,A,S,G,E,Π（UAS-AIOS） |
| **L1/L2/L3** | 自动执行 / 人工确认 / 禁止或双人审批 |
| **ChangeSet** | 可审计、可回滚的智能体变更契约 |
