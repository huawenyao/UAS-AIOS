认知超智能 UniAgent 标准化视图
枢纽 七层规范栈 三层认知模型 集群与协议流

七层依赖与横切引擎

上层依赖下层；通过下层一致性测试方可声称兼容。横切层贯穿全栈：适应、经济、可观测。展开每层查看规范要点与验收隐喻。

L0–L5 载荷与 Schema 刻意不标准化：内部推理与提示词

产品哲学对齐：标准化的是能力声明、状态外沿、消息语义与提交契约，使多智能体网络可替换、可测试、可审计——与「器」层实现解耦。

完整规范与 JSON 示例见文稿 UACA_UniAgent_TOGAF_Architecture.md。

L5 · World Commit 共享图写入
propose → validate → commit/reject；版本向量与 grounding_chain；接地一致性校验。唯一写共享世界的通道。
L4 · Network Policy 信任与隔离
通信权限、速率、路由、隔离域；trust_level 约束；断路器状态机。
L3 · Messaging 语义与背压
消息类型、超时、重试、幂等键、错误码；流控与协议一致性测试套件。
L2 · State Envelope 有界摘要
有界状态摘要、信息价值与陈旧度；不含私有 LLM 全上下文；序列化 round-trip 与大小阈值。
L1 · Capability Manifest 类型化能力
机器可读能力列表；capability_pipeline 组合约束；静态扫描与管道合法性。
L0 · Identity & Lifecycle 时钟与生命周期
unit_id、版本、心跳；混合物理/逻辑时钟；lifecycle_state 与信任级别。

横切（★）

  • Adaptation Engine — 在线学习与路由策略（如 bandit 选路）。
  • Economics Engine — 信用、信誉、定价与防搭便车。
  • Observability — 指标、告警、审计与归因。
