
# ΠPaw 高保真原型设计（可直接交付开发/Figma导出）
## 一、整体设计规范
### 1.1 设计理念
> 「极简专业、智能感知、数据驱动、层级清晰」
- 面向企业级用户，弱化娱乐化元素，突出专业性和可信度
- 智能能力前置，减少用户操作成本，信息主动找人
- 数据可视化突出重点，核心信息一目了然
- 严格匹配组织层级的信息权限和使用场景
### 1.2 视觉规范
| 类别 | 规范细节 | 数值/示例 |
|------|----------|-----------|
| **主色调** | 专业深蓝，传递可信、稳重的企业级产品气质 | 主色：`#165DFF`（阿里云蓝）<br> hover态：`#4080FF` <br> 点击态：`#0E42D2` |
| **辅助色** | 用于状态标识、分类展示 | 成功/正常：`#00B42A` <br> 告警/异常：`#FF7D00` <br> 错误/危险：`#F53F3F` <br> 信息/提示：`#86909C` |
| **中性色** | 用于文本、背景、边框 | 一级文本：`#1D2129` <br> 二级文本：`#4E5969` <br> 三级文本：`#86909C` <br> 边框/分割线：`#C9CDD4` <br> 背景色：`#F2F3F5` / `#F7F8FA` <br> 纯白背景：`#FFFFFF` |
| **字体体系** | 统一使用无衬线字体，清晰易读 | 中文：思源黑体 / PingFang SC <br> 英文/数字：Roboto / Helvetica Neue <br> 字号层级：12px（辅助文字）、14px（正文）、16px（小标题）、20px（大标题）、24px+（页面标题） <br> 字重：400（常规）、500（中等）、600（加粗） |
| **圆角规范** | 区分不同组件的圆角层级 | 卡片/弹窗圆角：`8px` <br> 按钮/输入框圆角：`6px` <br> 标签/徽章圆角：`4px` <br> 数据可视化组件圆角：`4px` |
| **阴影规范** | 分层阴影表达层级关系 | 卡片阴影：`0 2px 8px 0 rgba(0, 0, 0, 0.08)` <br> 弹窗阴影：`0 4px 16px 0 rgba(0, 0, 0, 0.12)` <br> 悬浮元素阴影：`0 8px 24px 0 rgba(0, 0, 0, 0.16)` |
### 1.3 组件体系
基于Ant Design 5.x企业级组件库扩展，统一组件交互逻辑：
- 表格、表单、弹窗、导航等基础组件严格遵循Ant Design规范
- 数据可视化组件基于ECharts定制，统一配色和交互逻辑
- 智能组件（AI分析卡片、异常告警卡片、推荐卡片等）定制开发，保持风格统一

## 三、全局交互规范（所有角色通用）
### 3.1 智能告警交互
- 告警分级：红色🔴高优（立即处理）、橙色🟠中优（24小时内处理）、黄色🟡低优（72小时内处理）
- 通知方式：桌面弹窗+声音提醒（高优）、消息中心通知（中低优）、短信/企业微信通知（高优且用户未读10分钟后）
- 处理入口：告警通知点击直接进入处理页面，自动加载所有上下文信息，不需要用户重新查数据
### 3.2 下钻分析交互
- 所有数据指标支持三级下钻：汇总数据→维度拆分→明细数据
- 下钻过程自动保留上下文，支持返回上一级
- 任意节点支持自动生成分析报告，导出为PPT/Excel/PDF
### 3.3 语音交互
- 全页面支持语音唤醒（唤醒词：「小P小P」）
- 支持自然语言查询、指令下达、报告生成
- 语音识别准确率≥98%，支持行业术语、方言识别
### 3.4 AI助手交互
- 所有页面右下角固定AI助手入口，点击唤起
- AI助手自动感知当前页面上下文，提问不需要重复描述场景
- 支持多轮对话，上下文自动继承
