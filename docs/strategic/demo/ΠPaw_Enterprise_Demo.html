ΠPaw助手
更换轴承需要注意以下几点：
1. 安装前要检查新轴承是否完好，型号是否匹配
2. 安装时要使用专用工具，禁止直接敲击轴承外圈
3. 轴承要安装到位，和轴肩贴合无间隙
4. 添加适量的专用润滑油，不要加过多
5. 安装后要手动转动检查是否灵活，有无异响

快速检查轴承的方法： 1. 看：外观有无裂纹、磕碰、锈蚀 2. 转：用手转动轴承，看是否转动灵活，有无卡顿异响 3. 晃：用手晃动轴承外圈，看是否有过大的间隙 4. 测：用游标卡尺测量轴承内径和外径，确认和旧轴承一致。
