### 1.2 认知智能的核心能力

| 能力 | 描述 | 与传统 AI 区别 |
|------|------|--------------|
| **语义理解** | 理解字面下的真实含义 | 不仅是关键词/语法 |
| **常识推理** | 运用人类常识判断 | 不仅是大数据统计 |
| **长期记忆** | 跨会话积累知识 | 不仅是对话上下文 |
| **自主规划** | 分解目标、制定步骤 | 不仅是指令执行 |
| **持续学习** | 从经验中自我提升 | 不仅是微调训练 |
| **意图理解** | 理解真正想要什么 | 不仅是指令解析 |
